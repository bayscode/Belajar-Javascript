bit = binary digit
| => adalah satuan terkecil dari data yang ada di dalam komputer
Nilai bit hanya 0 / 1
---------------------
Digit 1 itu terjadi ketika suatu komponen didalam komputer yang namanya transistor dialiri listrik, maka terjadilah bit 1, dan bit 0 terjadi ketika transistor tersebut tidak dialiri oleh listrik.
seperti ON = 1, OFF = 0.

Bilangan Desimal = 0 - 9
|=> Satuan, Puluhan, Ratusan, Ribuan, dst ...
